## ✨ Features

- **Modern & Premium Design:** Sleek dark mode styling with vibrant accents, glassmorphic elements, and radial glow effects.
- **Custom Interactive Cursor:** Smooth custom cursor and tracking ring with magnetic hover effects over interactive elements.
- **Scroll Animations:** Fluid fade-in animations on scroll utilizing high-performance `IntersectionObserver`.
- **CSS Grid & Flexbox:** Fully responsive, modern layouts that adjust beautifully from mobile screens to large desktop monitors.
- **Pure Vanilla Stack:** Built strictly with HTML5, CSS3, and modern ES6 JavaScript. No framework overhead or heavy dependencies.

## 🚀 Getting Started

Since the project is built using pure frontend web standards, running it is extremely straightforward:

### Option 1: Direct Open
Simply double-click the `portfolio.html` file to open it in your default web browser.

### Option 2: Local Development Server (Recommended)
To prevent potential CORS issues or local resource loading limits, run a simple local web server.

**Using Python:**
```bash
# Python 3
python -m http.server 8000
```
Then open `http://localhost:8000/portfolio.html` in your browser.

**Using Node.js (serve):**
```bash
npx serve .
```
